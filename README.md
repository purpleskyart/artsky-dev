# artsky-dev

This is the **development deployment** of [artsky](https://github.com/slrgt/artsky).

## What is this?

artsky is a **PWA** (Progressive Web App) that provides an app-style view for [Bluesky](https://bsky.app) (AT Protocol). It features:
- Feed with masonry grid of images and videos
- Artboards to save posts
- Comments and replies
- Works as a native app on phone/desktop

## Development vs Production

- **This dev site** (`artsky-dev`) - Testing new features before they go live
- **Live site** (`artsky`) - Stable production version

The development branch allows testing changes without affecting the live app. Once features are tested and approved here, they get merged to the main branch for production deployment.
